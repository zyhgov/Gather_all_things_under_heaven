在亚洲引入数据驻留

数据驻留建立在 OpenAI 的企业级数据隐私、安全性和合规性计划之上，
为全球客户提供支持。

我们宣布 ChatGPT Enterprise、ChatGPT Edu 和 API 平台在日本、印度、新加坡和韩国进行数据驻留。这有助于在这些国家/地区运营的组织在业务中使用 OpenAI 产品并使用 AI 构建新的解决方案时满足当地数据主权要求。

数据驻留建立在 OpenAI 现有的企业级数据隐私、安全性和合规性功能之上。

API、ChatGPT Enterprise 和 ChatGPT Edu 在亚洲的数据驻留

产品名称 支持国家/地区 数据驻留功能
ChatGPT Enterprise 日本、印度、新加坡、韩国 支持数据驻留
ChatGPT Edu 日本、印度、新加坡、韩国 支持数据驻留
API 日本、印度、新加坡、韩国 支持数据驻留

通过数据驻留，符合条件的 API 客户以及新的 ChatGPT Enterprise 和 Edu 客户可以选择将客户内容静态存储在受支持的国家/地区。

  • API 客户可以启用数据驻留功能。
  • ChatGPT Enterprise 客户可以配置数据驻留选项。
  • ChatGPT Edu 客户可以使用数据驻留服务。
  1. 第一步：注册账户。
  2. 第二步：选择数据驻留区域。
  3. 第三步：开始使用 OpenAI 产品。
